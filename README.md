# Editable Chatbot

## Overview
This project is an editable chatbot application that allows users to interact with various AI models.

## Setup Instructions

### Prerequisites
- Python 3.7 or higher
- Git

### 1. Clone the Repository
First, clone the repository to your local machine:
```bash
git clone https://github.com/emanuele-caddeo/editable-chatbot.git
cd editable-chatbot
```

### 2. Install Python Dependencies
Make sure you have a virtual environment set up. You can create one using:
```bash
python -m venv .venv
```
Activate the virtual environment:
```bash
# On Windows
.venv\Scripts\activate
```
Then, install the required Python packages:
```bash
pip install -r requirements.txt
```

### 3. Run the APP
```bash
python run_app.py
```


### 5. Access the Client
If the client doesn't automatically start, you can access the application in your web browser at `http://localhost:5000`.
